Exercícios teóricos

5- Para que serve o comando "alert" em Javascript?

Resposta: Ele emite um alerta em formato de pop-out para o navegador, que vai mostrar para o usuário uma caixinha com as opções de cancelar ou dar ok.


6- Para que serve o comando "prompt" em Javascript?

O prompt serva para ler os dados que o usuário deseja e precisa para processar qualquer operação. Enquanto o alert indica a saída de dados, o prompt indica a entrada dos mesmos.


7- Para que serve o comando "console.log()" em Javascript?

Serve para dizer algo para o usuário também, mas no console, essa mensagem aparecerá no terminal do site, ao abrir as ferramentas de desenvolvedor.
